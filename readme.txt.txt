Rick and Morty Character SPA
این SPA یک نمونه ساده از یک وب اپلیکیشن تک صفحه‌ای (Single Page Application) با استفاده از ReactJS، Apollo Client برای انجام کوئری به یک GraphQL سرور، Redux برای مدیریت وضعیت برنامه، و ant design برای استایلینگ و طراحی واسط کاربری (UI) می‌باشد. این SPA اطلاعات کاراکترهای مجموعه "Rick and Morty" را از یک منبع خارجی با استفاده از GraphQL می‌گیرد و نمایش می‌دهد.

نحوه اجرا
نصب وابستگی‌ها:
ابتدا وابستگی‌های پروژه را نصب کنید. از دستور زیر استفاده کنید:
npm install

اجرای برنامه:
پس از نصب وابستگی‌ها، برنامه را اجرا کنید:
npm start

سپس می‌توانید برنامه را در مرورگر خود باز کنید. وب اپلیکیشن در آدرس http://localhost:3000 در دسترس خواهد بود.


قابلیت‌ها
نمایش لیستی از کاراکترهای مجموعه "Rick and Morty" با اطلاعات مانند نام و نوع
استفاده از GraphQL برای انجام کوئری به منبع اطلاعاتی.
مدیریت وضعیت برنامه با Redux.
طراحی واسط کاربری با استفاده از ant design.

منابع
React
Apollo Client
Redux
ant design
